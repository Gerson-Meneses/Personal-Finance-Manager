
```sh
bun run dev
```
Versionado 1 de app para llevar un registro claro de finanzas y ayudar con un presupuesto

* Cuentas:
Se crea una cuenta de tipo efectivo al crear al usuario, la cual no se puede eliminar, ni editar
El usuario puede crear cuentas, las cuales por ahora solo puede cambiar el nombre una vez creadas
Si desea modficar otra cosa se recomienda eliminar y crear nuevamente, antes de agregar transacciones a la misma, ya que tampoco se podra eliminar si tiene transacciones.

Categorias:
Se crean cateogrias base, no editables, ni eliminables, solo se puede cambiar la visibilidad.
Las que crea el usuario, se 
open http://localhost:3000
